# PlagGuard - Originality Verification System

A full-stack web application for product verification and user authentication.

## Project Structure
- **backend/**: Node.js/Express server with MongoDB integration.
- **frontend/**: HTML, CSS, and Vanilla JavaScript for the user interface.

## Setup
1. Go to `backend/`
2. Run `npm install`
3. Update `backend/.env` with your MongoDB URI.
4. Run `node server.js`
5. Open `http://localhost:5000/login.html`

## Features
- User Signup/Login (Email, Phone, Google)
- Portfolio/Landing pages
- MongoDB Data Persistence
- Responsive Design
